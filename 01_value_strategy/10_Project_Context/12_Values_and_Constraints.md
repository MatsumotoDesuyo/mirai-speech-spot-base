# Values and Constraints

## Core Values
*  ナレッジを独占しない。OSS化を前提とする。
*  他陣営の妨害に繋がる機能は作らない。
*  公職選挙法、道路交通法、その他法令を遵守する。

## Content & Licensing
*  **オープンソースライセンス**: Apache-2.0からAGPL-3.0へ移行する。（※現在移行準備中）
*  **コントリビューション**: 外部からのコントリビューションを受け入れる際は、将来的なライセンス変更の柔軟性を担保するため、CLA（Contributor License Agreement）の締結を必須とする。
