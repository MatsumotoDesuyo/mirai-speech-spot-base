# 11_NorthStar.md

## NothStar Identity (究極の北極星)
**「新人陣営が『まるで10年目のベテランのように』迷いなく振る舞える状態の創出（Zero-friction Experience）」**

プロジェクトのあらゆる決断は、この定性的な理想像に帰結する。
「はじめての立候補でも、地図を開けば5秒で明日の10箇所の演説ルートが組める」。この圧倒的な「摩擦のなさ」こそが、既存政党との参入障壁を破壊する刃となる。

## Core Value (提供する中核価値)
**「インフラが生み出した『政治活動の可処分時間』（Time Saved）」**

新人の最大の障壁は「非生産的な調査・悩みの時間」にある。
「どこで演説できるか」「車は停められるか」といった調査に奪われていた時間を極限までゼロに近づけ、その浮いた全時間を、本来の政治活動（有権者との対話）に再投資させる。これこそが我々のインフラが提供する本質的価値である。

## Key Metrics (計測可能な目標指標)

> [!NOTE]
> 以下のKey Metricsは策定中（仮置き）の指標である。今後チームメンバーとの議論を経て正式に決定される。

上記の「価値」が実際に届いているかを測るため、以下の定量指標を各フェーズ（開発・運用）で追跡する（案）。

### 1. Data Quality & Coverage (インフラの準備状態)
*   **即戦力データの網羅率**: 特定エリアにおける「推奨Lv ◯以上 ＋ 詳細情報（写真・車可否）あり」の、そのまま使える高品質スポットの数とカバー率。

### 2. Usage & Action (価値の行使)
*   **実働への変換数（Actionable Insights Used）**: アプリから「URLがシェアされた回数」や「道案内が起動された回数」など、データが実際の「現地の行動」に変換された総数。

### 3. Velocity (UXの到達度)
*   **意思決定速度（Time to Action）**: アプリを開いてから「よし、ここに行こう」と意思決定（＝URLコピー等のアクション）するまでの平均時間の短縮。迷うことなくサクサク決まる状態を数値化する。
