# Directory Structure Standard

## 1. 目的 (Purpose)

C2V Engineで利用される基本的なとドキュメントを標準化し、プロジェクトにおける情報の透明性とアクセス性を最大化する。
このドキュメントには、C2V Engineが必要とする汎用的なディレクトリ情報についてのみ記載し、プロジェクト毎独自のディレクトリ情報は記載しない。

## 2. ディレクトリ構造 (Directory Structure)

### 2.1. `/00_C2V`

このディレクトリには、C2V Engineフレームワークの思想的根幹を定義する、普遍的なドキュメントが格納される。

### 2.2. `/90_roles`

このプロジェクトで定義された、全てのロール（仮面）の定義書が格納される。

### 2.3. `/91_standardized_procedures`

AIによるガイドを想定した、標準的な手順書（プロシージャ）が格納される。
