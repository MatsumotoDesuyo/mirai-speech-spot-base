# SpotPosting（スポット投稿）

## 概要
ユーザーが新しい演説スポットを地図上に登録する機能。

## 対象ユーザー
Campaign Staff (Supporter)

## トリガー
地図上で長押し（モバイル）または右クリック（デスクトップ）。

## 入力時の注意事項
以下の2点が明確になるよう、写真や説明文で情報を入力すること：
1. **候補者がどこに立って演説できるか**
2. **選挙カーをどこに停めるか**

写真がなくても登録可能だが、上記が伝わるよう説明文を詳しく記載すること。

## 入力項目

| 項目 | 必須 | 説明 |
|:---|:---:|:---|
| タイトル | ✅ | 場所の名前（例：〇〇スーパー前） |
| 写真 | - | 現場の状況がわかる写真。撮影ガイダンス表示あり。※仮運用として写真なしでの登録を許可中 |
| 推奨Lv（Speech Score） | ✅ | 1〜10のスライダー。デフォルト: Lv 7 |
| 聴衆属性 | - | マルチセレクト（主婦、学生、社会人、高齢者、ファミリー） |
| 選挙カー利用可否 | ✅ | 使用可 / 一瞬の乗降のみ可 / 不可 |
| おすすめ時間帯 | - | 1時間単位のマルチセレクト（08:00〜19:00） |
| 説明 | - | 補足情報 |
| パスコード | ✅ | 共有パスコード（チーム内で共有） |

### 推奨Lvの目安
- **Lv 10:** 【S級】広場・ランドマーク（確実な聴衆）
- **Lv 8-9:** 【A級】主要駅・スーパー（主力）
- **Lv 6-7:** 【B級】商店街・公園（堅実な対話）
- **Lv 5:** 【C級】穴場（特定層向け）
- **Lv 1-4:** 推奨度低

### 時間帯の制約
公職選挙法により、マイクを使った演説は8時〜20時のみ許可（20時から開始はできない）。
