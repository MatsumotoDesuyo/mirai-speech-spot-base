# 🗺️ みらい演説スポットBase (Mirai Speech Spot Base)

**「日本の政治参入における『知的資産による障壁』を限りなくゼロに近づけ、志ある誰もが挑戦できるインフラを創る」**

みらい演説スポットBaseは、新規の政治参入者が街頭演説を行うためのナレッジ（場所、時間帯、聴衆属性、車両の可否など）を蓄積・共有するためのマップアプリケーションです。これまでベテラン陣営の暗黙知とされていた情報をオープンにし、誰もが「まるで10年目のベテランのように」演説活動を行えるプラットフォームを目指しています。

---

## 🎯 プロジェクトの思想：The C→V Engine
本プロジェクトは、人間の創造性を価値に変換するためのフレームワーク「**C2V Engine**」を採用しています。
すべての開発プロセス・意思決定は、個人の好みを排除し、定められた**ミッション**と**北極星（North Star）**にのみ従います。
詳細な思想やルールについては、プロジェクトルートの `00_C2V` ディレクトリのドキュメントに定義されています。
